📌 .gitignore 파일 설명

이 문서는 .gitignore 파일의 각 항목을 설명하여, 특정 파일/폴더가 제외되는 이유를 명확히 합니다.

1. 사용자 설정 파일 (User Settings)

제외 파일/폴더 설명
*.suo 솔루션 사용자 설정 파일
*.user 사용자별 프로젝트 설정 파일
*.userosscache 사용자 설정 캐시 파일
*.sln.docstates 솔루션 문서 상태 저장 파일
*.userprefs MonoDevelop/Xamarin Studio 사용자 설정 파일

2. 빌드 결과물 (Build Outputs)

제외 파일/폴더 설명
[Dd]ebug/ Debug 빌드 폴더
[Rr]elease/ Release 빌드 폴더
[Bb]in/ 실행 파일 및 라이브러리 저장 폴더
[Oo]bj/ 객체 파일(obj) 저장 폴더
x64/, x86/ 64비트 및 32비트 빌드 폴더
[Ll]og/ 로그 파일 폴더

3. 개발 도구 관련 파일

제외 파일/폴더 설명
.vs/ Visual Studio 캐시 및 설정 파일
*.VC.db, *.VC.VC.opendb Visual C++ 빌드 캐시
*.sdf, *.ncb Visual Studio 데이터베이스 파일
*.aps, *.cachefile Visual C++ 캐시 파일
*.pidb MonoDevelop 프로젝트 인덱스 파일

4. 패키지 관리자 관련 파일

제외 파일/폴더 설명
packages/ NuGet 패키지 폴더
.nuget.props, .nuget.targets NuGet 빌드 설정 파일
node_modules/ Node.js 패키지 폴더
.paket/, paket-files/ Paket 패키지 관리 폴더

5. 로그 및 임시 파일

제외 파일/폴더 설명
*.log 로그 파일
*.tmp 임시 파일
*.cache 캐시 파일
*.pidb 프로젝트 인덱스 파일
*.svclog 서비스 로그 파일

6. 데이터베이스 파일

제외 파일/폴더 설명
*.mdf, *.ldf SQL Server 데이터베이스 파일
*.dbproj.schemaview 데이터베이스 프로젝트 파일

7. 테스트 파일

제외 파일/폴더 설명
[Tt]est[Rr]esult*/ 테스트 결과 파일
TestResult.xml NUnit 테스트 결과 파일
*.VisualState.xml Visual Studio 테스트 상태 파일

8. 배포 및 Azure 관련 파일

제외 파일/폴더 설명
publish/, *.publishproj Click-Once 배포 폴더
*.publishsettings, *.pubxml 웹 배포 설정 파일
csx/, *.build.csdef Azure 빌드 출력 폴더
AppPackages/, BundleArtifacts/ Windows Store 앱 패키지 폴더

9. 프로젝트 업그레이드 및 백업 파일

제외 파일/폴더 설명
_UpgradeReport_Files/ 프로젝트 업그레이드 보고서
Backup*/ 프로젝트 백업 파일
UpgradeLog*.XML, UpgradeLog*.htm 업그레이드 로그 파일

10. DLL 파일 및 기타 제외할 파일

제외 파일/폴더 설명
*.dll 동적 링크 라이브러리 파일
*.bmp 이미지 파일
.idea/ JetBrains Rider 설정 파일
.cr/ CodeRush 설정 파일
_CudaGPUInfo/ Cuda GPU 정보 파일
Generated_Code/ 자동 생성된 코드 폴더

🔍 설명 파일 사용 방법

  1. .gitignore 파일은 프로젝트의 불필요한 파일을 버전 관리에서 제외하기 위해 사용됩니다.
  2. 이 문서를 참조하여 .gitignore 항목이 제외된 이유를 확인할 수 있습니다.
